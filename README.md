SQL Sales Analysis

Built a relational database from scratch using SQLite to analyze guest orders and revenue across a hotel services menu.

Database structure:
Three tables — customers, products, and orders — linked together using JOIN queries.

Key findings:

Top revenue product: Diner Buffet at 935.00

Top 3 customers by spend: Verlinden (600), Vansteenbergen (595), Birenbaum (450)

Tools used: SQLite, DB Browser for SQLite
